# Smart Study Desk Software

## Project Description
Smart Study Desk Software is a comprehensive planning and productivity application designed to help students and professionals manage their schedules efficiently. The software integrates various productivity tools to create a centralized workspace for all academic and work-related activities.

## Features
- **Task Management**: Create, update, and manage tasks and deadlines.
- **Scheduling**: View and schedule studying sessions and reminders.
- **Analytics**: Track productivity over time and adjust plans accordingly.
- **Collaboration**: Share tasks and schedules with peers.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Heroku

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/baargavib/SmartStudyDesk-Software.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SmartStudyDesk-Software
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## How to Run the Application
1. Start the server:
   ```bash
   npm start
   ```
2. Open `http://localhost:3000` in your browser to access the application.
